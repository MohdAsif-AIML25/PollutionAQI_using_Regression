# PollutionAQI_using_Regression
Air Quality Index (AQI) Prediction Using Regression

This project focuses on analyzing and predicting Air Quality Index (AQI) across Indian cities using machine learning regression techniques. The goal is to understand how different air pollutants contribute to AQI and to build predictive models that can estimate air quality levels accurately.

The project uses a real-world air pollution dataset containing pollutant concentrations such as PM2.5, PM10, NO₂, CO, SO₂, and O₃. After cleaning and preprocessing the data, Linear Regression, Ridge Regression, and Lasso Regression models are implemented and compared to evaluate performance and understand feature importance.

In addition to prediction, the project includes exploratory data analysis (EDA) and visualization, such as correlation heatmaps, AQI distribution plots, and a state-wise AQI map of India using GeoPandas. These visualizations help translate raw data into meaningful environmental insights.

This repository demonstrates an end-to-end machine learning workflow, including data preprocessing, model training, evaluation, and visualization, making it suitable for data science portfolios, academic projects, and interview discussions.
